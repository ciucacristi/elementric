FontAwesome Icons

Elementric components - just copy the HTML code of an element and paste it into your code (after you read the installation guide).

Icons

Icons used are from FontAwesome Icons

How to use the icons?

Step 1

Choose a icon from the Icons List and copy the HTML tag

Step 2

Paste the HTML icon tag to your code

Example:

         

         
  <i class="fas fa-cat fa-3x"></i>
