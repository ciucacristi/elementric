CDN installation with CSS and JS files

To use the Elementric components, you need to add the CSS and JS files into your project HTML code. You can do this by
copying the below Elementric files links and adding in your HTML code:

      

      
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/ciucacristi/elementric/Elementric/css/elementric.css"> <!-- CSS -->
      
    
      

      
  <script src="https://cdn.jsdelivr.net/gh/ciucacristi/elementric/Elementric/js/elementric.js"></script> <!-- JavaScript -->
